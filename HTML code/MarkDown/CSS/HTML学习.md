# HTML学习

## 标签属性

##### 属性有对应的属性值和属性名，如：

```
//滚动效果 <marquee loop="1" bgcolor="orange">what can i say!</marquee>
//输入<input type="password">
```

##### 有些属性没名，只有值，如：

```
<input disabled>
```

##### 属性名不得重复，否则按先后顺序执行，如：

```
<input type="text" type="password">//优先执行text
```
